# 「新员工入职卡片」项目执行方案

**汇报人：行政部**  
**汇报日期：2026年7月10日**  
**项目版本：v3.0**

---

## 一、项目背景

### 1. 公司背景

元保（Yuanbao Inc.）是国内领先的互联网保险科技平台，2025年4月在美国纳斯达克正式上市，累计服务超千万家庭，定位「持牌保险专家」。

公司内部员工以「保贝」互称，组织文化强调科技驱动、高效协同。本次入职卡片项目，将作为新员工入职第一天接触公司文化和行政服务的重要触点。

### 2. 项目背景

目前新员工入职时会收到公司品牌文件袋，内含 A5 笔记本、签字笔、便利贴和鼠标垫等物料，但缺少一张能够同时承载「欢迎仪式感」「企业文化传达」「行政服务引导」的入职卡片。

因此，本项目计划在新员工文件袋中新增一张 A5 入职卡片，一面用于欢迎新员工并传递企业文化，另一面用于推广「元保AI行政服务台」。

---

## 二、项目价值

| 价值维度 | 具体价值 |
|---|---|
| 对新员工 | 入职第一天即可感受到公司欢迎和行政服务温度，降低陌生感 |

| 对组织协同 | 将行政服务入口前置到入职流程中，提升服务触达率和使用习惯 |
| 对部门形象 | 体现行政部门从被动响应到主动服务设计的转变 |

## 四、卡片内容方案

### 1. 正面内容：欢迎 + 企业文化

正面重点承担「欢迎新员工」和「传递公司文化」的功能。

主要内容包括：

- 元保品牌标识
- 「欢迎加入元保」主标题
- 「很高兴与你成为保贝」欢迎语
- 公司使命：以科技力量，让每个人都能获得安心保障
- 核心价值观：用户至上、科技驱动、简单高效、合作共赢、拥抱变化、诚信担当
- NASDAQ 上市标识
- 入职日期填写区

### 2. 背面内容：元保AI行政服务台推广

背面重点承担「行政服务入口引导」的功能，帮助新员工快速知道：遇到行政、IT等问题时，应该先去哪里问。

主要内容包括：

- 标题：元保AI行政服务台
- 服务说明：7x24 小时在线，覆盖行政、IT等全场景，答你所问
- 使用方式：飞书中搜索「元保AI行政服务台」开启对话
- 服务范围：员工福利、IT服务、物品领用、出行停车、报销相关、访客接待、快递服务、母婴室、报修服务
- 示例问题：WiFi密码、怎么报销、工位报修、办公用品哪里领、停车位怎么办理等
- 公司官网：yb-inc.com

## 五、设计方向

本次卡片设计采用公司品牌色体系，整体风格保持专业、克制、清晰，避免过度花哨。

| 设计项 | 设计说明 |
|---|---|
| 主色 | 深蓝、品牌蓝，体现专业和可信赖感 |
| 强调色 | 品牌金，用于重点信息和视觉点缀 |
| 字体 | 使用 PingFang SC / Microsoft YaHei 等中文友好字体 |
| 版式 | A5 双面设计，信息分区清晰，方便阅读 |
| 风格 | 简洁商务，不做过度装饰，适合企业正式物料 |
| 印刷建议 | 300g 铜版纸，双面彩印，覆哑膜 |

## 八、风险与应对

| 风险 | 影响 | 应对方案 |
|---|---|---|
| 企业文化文案未及时确认 | 影响设计定稿和印刷周期 | 提前与 HR 确认，保留简化版文案作为备选 |
| 服务台入口名称变更 | 影响新员工搜索准确性 | 正式印刷前与 IT 再次确认飞书搜索名称 |
| 印刷颜色与设计稿存在偏差 | 影响实物质感 | 先打样再批量印刷，确认色彩后再生产 |
| 新员工使用率不达预期 | 影响项目效果 | 在入职培训中增加 2 分钟服务台使用引导 |
| 卡片内容后续需要更新 | 影响长期复用 | 保留可编辑设计源文件，后续按季度迭代 |
